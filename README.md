# html-to-pdf

Converting HTML template to PDF files

[![LICENSE](https://img.shields.io/github/license/henriquecarv/html-to-pdf.svg)](./LICENSE)

## System Requirements

- [Node.js](https://nodejs.org)

## Installing

- `npm install`

## Creating POC

- `npm start`

The created files will be saved on `~/output`
